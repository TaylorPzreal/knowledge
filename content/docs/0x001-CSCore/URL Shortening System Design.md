## 一、Introduction
A URL shortener is a service that takes a long URL and returns a **shorter**, **unique alias** that redirects to the original URL.


## Kutt.it
**Kutt** is a modern URL shortener with support for custom domains. Create and edit links, view statistics, manage users, and more.
