## 一、Introduction

### 软件质量体系
KISS原则
1. 学习上手成本低
2. 不会因为操作失误搞挂程序
3. 开发体验
	- 可维护性
	- 可靠性
	- 易用性

### 软件工程方法论
#### 设计模式
1. 观察者模式：不同对象之间通信
	- EventEmitter
	- DOM addEventListener
2. 外观模式：兼容各个不同的实现，提供统一接口
	- jQuery

#### 设计模式六大法则

2. 里氏替换原则
3. 依赖倒置原则
4. 接口隔离

6. 开闭原则
